# Cybersecurity Writeups Repository

Welcome! 👋  
This repository contains my personal write-ups from various cybersecurity labs and challenges.  
It is organized by platform (TryHackMe, HackTheBox) and by topic.

**Disclaimer:**  
All content here is for **educational purposes only**. No real targets or illegal actions are included.  
Follow the law and use this knowledge ethically.

---

## Repo Structure

- `TryHackMe/` – Labs from TryHackMe
- `HackTheBox/` – Labs from HackTheBox
- `Templates/` – Markdown templates for write-ups and lab notes
- `README.md` – this file

---

## How I Organize Write-ups

Each lab has its own folder with:
- `README.md` → main write-up
- `screenshots/` → optional screenshots
- `files/` → optional captures (pcap, outputs)

---

## Write-up Template

Inside `Templates/writeup-template.md`:

```markdown
# Title

## Goal
- Describe the lab/challenge objective

## Recon / Setup
- How you prepared / environment info

## Steps Taken
- Step-by-step explanation (commands, screenshots, outputs)

## Evidence
- Screenshots, captures, tables

## Analysis
- What happened and why (include network/protocol explanation)

## Mitigation / Defensive Notes
- How this could be defended against in real-world networks

## Lessons Learned
- Personal takeaways
